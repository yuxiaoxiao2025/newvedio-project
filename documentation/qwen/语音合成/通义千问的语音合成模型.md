# 通义千问的语音合成模型

通义千问的语音合成提供多种拟人音色，支持多语言及方言，并可在同一音色下输出多语言内容，系统可自适应语气，流畅处理复杂文本。

## 快速开始（Python）

说明：DashScope Python SDK 中的 `SpeechSynthesizer` 接口已统一为 `MultiModalConversation`，仅需替换接口名称即可，其他参数兼容。

关键参数：

- `text`：指定文本
- `voice`：指定音色
- `language_type`：指定语种，可选 Chinese、English、German、Italian、Portuguese、Spanish、Japanese、Korean、French、Russian、Auto（默认）

同步合成并获取 URL：

```python
#  DashScope SDK 版本不低于 1.24.6
import os
import dashscope

# 北京地域；如使用新加坡地域，替换为：https://dashscope-intl.aliyuncs.com/api/v1
dashscope.base_http_api_url = 'https://dashscope.aliyuncs.com/api/v1'

text = "那我来给大家推荐一款T恤，这款呢真的是超级好看，这个颜色呢很显气质，而且呢也是搭配的绝佳单品，大家可以闭眼入，真的是非常好看，对身材的包容性也很好，不管啥身材的宝宝呢，穿上去都是很好看的。推荐宝宝们下单哦。"

# SpeechSynthesizer接口使用方法：dashscope.audio.qwen_tts.SpeechSynthesizer.call(...)
response = dashscope.MultiModalConversation.call(
    model="qwen3-tts-flash",
    # 若未配置环境变量，请替换为：api_key = "sk-xxx"
    api_key=os.getenv("DASHSCOPE_API_KEY"),
    text=text,
    voice="Cherry",
    language_type="Chinese",
    stream=False
)
print(response)
```

## 实时播放（Python，流式输出）

```python
#  DashScope SDK 版本不低于 1.24.6
# pyaudio 安装见各平台说明

import os
import dashscope
import pyaudio
import time
import base64
import numpy as np

dashscope.base_http_api_url = 'https://dashscope.aliyuncs.com/api/v1'

p = pyaudio.PyAudio()
stream = p.open(format=pyaudio.paInt16,
                channels=1,
                rate=24000,
                output=True)

text = "你好啊，我是通义千问"
response = dashscope.MultiModalConversation.call(
    api_key=os.getenv("DASHSCOPE_API_KEY"),
    model="qwen3-tts-flash",
    text=text,
    voice="Cherry",
    language_type="Chinese",
    stream=True
)

for chunk in response:
    if chunk.output is not None:
        audio = chunk.output.audio
        if audio.data is not None:
            wav_bytes = base64.b64decode(audio.data)
            audio_np = np.frombuffer(wav_bytes, dtype=np.int16)
            stream.write(audio_np.tobytes())
    if chunk.output is not None and chunk.output.finish_reason == "stop":
        print("finish at:", getattr(getattr(chunk.output, 'audio', None), 'expires_at', None))

time.sleep(0.8)
stream.stop_stream()
stream.close()
p.terminate()
```

文本生成模型基于提示词（Prompt）生成连贯文本，适用于内容创作、客服问答、翻译与摘要等场景。

## 核心概念

- 消息角色：`system`（人设与规范）、`user`（指令）、`assistant`（回复）
- 输入格式：`messages=[system, user]`；建议提供 `system` 以稳定输出

## 快速开始（OpenAI 兼容）

```python
import os
from openai import OpenAI

client = OpenAI(
    api_key=os.getenv("DASHSCOPE_API_KEY"),
    base_url="https://dashscope.aliyuncs.com/compatible-mode/v1",
)

completion = client.chat.completions.create(
    model="qwen-plus",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "你是谁？"},
    ],
)
print(completion.choices[0].message.content)
```

## 模型选型

- 通用：`qwen3-max`（复杂任务）、`qwen-plus`（均衡）、`qwen-flash`（快速低成本）
- 特定场景：代码、长文档、翻译、数据挖掘、角色扮演、深入研究等系列
- 多模态：`qwen3-vl-*`（文图）、`qwen-omni`（文音）
- 第三方：DeepSeek、Kimi、GLM 等

## 异步调用（OpenAI 兼容）

```python
import os
import asyncio
from openai import AsyncOpenAI
import platform

client = AsyncOpenAI(
    api_key=os.getenv("DASHSCOPE_API_KEY"),
    base_url="https://dashscope.aliyuncs.com/compatible-mode/v1"
)

async def task(q):
    r = await client.chat.completions.create(
        messages=[
            {"role": "system", "content": "You are a helpful assistant."},
            {"role": "user", "content": q}
        ],
        model="qwen-plus",
    )
    print(r.choices[0].message.content)

async def main():
    qs = ["你是谁？", "你会什么？", "天气怎么样？"]
    await asyncio.gather(*[task(q) for q in qs])

if platform.system() == 'Windows':
    asyncio.set_event_loop_policy(asyncio.WindowsSelectorEventLoopPolicy())
asyncio.run(main())
```

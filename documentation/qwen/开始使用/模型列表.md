## 模型总览

| 类别 | 子类别 | 说明 |
| --- | --- | --- |
| 文本生成 | 通用大语言模型 | 通义千问（商业版 Max/Plus/Flash、开源 Qwen3/2.5、超长文档 Long），第三方：DeepSeek、Kimi、GLM 等 |
| 多模态模型 | 视觉理解、视觉推理、音频理解、全模态、实时多模态 |
| 领域模型 | 代码、数学、翻译、法律、数据挖掘、深入研究、意图理解、角色扮演 |
| 图像生成 | 文生图：通义千问、通义万相、SD/FLUX；更多：创意海报、WordArt 锦书 |
| 图像编辑 | 通义千问图像编辑、通义万相图像编辑及翻译、涂鸦、局部重绘、人像风格重绘等 |
| 语音合成与识别 | Qwen-TTS(-Realtime)、CosyVoice、Sambert；ASR：Qwen-ASR(-Realtime)、Fun-ASR、Gummy、Paraformer、SenseVoice |
| 视频 | 文生视频、图生视频（首帧/首尾帧/多图）、通用视频编辑、口型替换、风格转换 |
| 向量 | 文本向量、多模态向量 |
